##Project Overview

This project analyzes customer retention behavior using cohort analysis on the Brazilian E-Commerce Public Dataset by Olist.

The objective is to understand customer retention patterns, repeat purchase behavior, and customer churn across different acquisition cohorts.

Tools

* Google BigQuery
* SQL
* Power BI
* Canva

Key Metrics

* Total Customers: 96,096
* Total Orders: 99,441
* Total Revenue: 16.01M BRL
* Repeat Purchase Rate: 3.12%
* Average Month 1 Retention Rate: 5.22%

Project Structure

1. Introduction to Cohort Analysis
2. Dataset Overview
3. Building the Cohort Retention Framework
4. Customer Retention Analysis
5. Business Insights and Recommendations

Key Findings

* Customer retention declined significantly after the first purchase.
* Most customers made only one purchase.
* Repeat purchasing behavior remained limited.
* Long-term customer retention was relatively low.

Dataset

Brazilian E-Commerce Public Dataset by Olist.
